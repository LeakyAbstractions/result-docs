# Demo Projects

To help you become familiar with this library, you can explore two demo projects that showcase how to handle and
serialize Result objects within popular frameworks like [**Spring Boot**][SPRING_BOOT] and [**Micronaut**][MICRONAUT].
Each project provides a working example of a "pet store" web service that exposes a REST API for managing pets. They are
based on [Swagger Petstore Sample][SWAGGER_PETSTORE] and you can interact with them using [**Swagger-UI**][SWAGGER_UI].
